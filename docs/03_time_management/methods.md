## 重要与紧急事项分类方法

!!! note "1.重要且紧急"

    - 紧急状况
    - 迫切的问题
    - 限期完成的工作
    - 你不做其他人也不能做

!!! note "2.重要不紧急"

    - 准备工作
    - 预防措施
    - 价值观的澄清
    - 计划
    - 人际关系的建立
    - 真正的再创造
    - 增进自己的能力

!!! note "3.紧急不重要"

    - 造成干扰的事、电话、信件、报告会议
    - 许多迫在眉捷的急事
    - 符合别人期望的事

!!! note "4.不重要不紧急"

    - 忙碌琐碎的事
    - 广告函件
    - 电话
    - 逃避性活动
    - 等待时间

> **优先顺序=重要性\*紧迫性**
>
> 在进行时间安排时，应权衡各种事情的优先顺序，要学会“弹钢琴”。
>
> 对工作要有前瞻能力，防患于未然，如果总是在忙于救火，那将使我们的工作永远处理被动之中。
